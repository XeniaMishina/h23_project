## KDM6A

## Описание 
- product: H3K27
- f: деметилирует "Lys-27" гистона H3, тем самым играя центральную роль в гистоновом коде
- комплекс: CHD8, MLL2/3, MLL4/WBP7, COMPASS-like MLL3,4
- ткани: костный мозг, щитовидная железа, эндометрий, яичники и др.
- домены: JmjC, TPR
